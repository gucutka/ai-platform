# Governance Handbook

## Knowledge Owners (project)

- Business Analyst → Business Layer
- Product Analyst → Product Layer
- Architect → Technical Layer + Design System

## Platform Governance

- **Platform Architect** — registry, platform ADR, versioning (not in project SDLC)
- **Platform Owner** — rollout, onboarding, GitHub App

## Human Governance (escalation)

- Tech Lead — merge, PR, agent failures
- Engineering Manager — budget, blocked SLA
- Security Owner — security blocks

`governance/architecture-governance.yaml`

- Architect Review Gate
- architecture-review-agent
- ADR lifecycle, exceptions, tech debt
